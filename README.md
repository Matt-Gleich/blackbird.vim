# blackbird.vim

🏴 blackbird theme for vim

## 🚀 Install

You can install using [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'Matt-Gleich/blackbird.vim'
```

Once you have it installed just set it as your theme:

```vim
colorscheme blackbird
```
